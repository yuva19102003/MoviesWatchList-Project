
Create a mobile-responsive Next.js single-page application for a **Movie Watchlist Web App**.

### Requirements:

1. Use **Next.js (App Router)** with **TailwindCSS** for styling.
2. Fetch API endpoints from URLs stored in `.env.local`:

   * `NEXT_PUBLIC_API_BASE_URL=http://localhost:5000`

* **GET /movies/{id}**

  * Response:

    ```json
    {
      "id": 1,
      "title": "Inception",
      "genre": "Sci-Fi",
      "director": "Christopher Nolan",
      "releaseYear": 2010,
      "watched": true,
      "rating": 9,
      "description": "A mind-bending thriller",
      "imageUrl": "https://image.com/inception.jpg"
    }
    ```

* **POST /movies**

  * Request:

    ```json
    {
      "title": "Interstellar",
      "genre": "Sci-Fi",
      "director": "Christopher Nolan",
      "releaseYear": 2014,
      "watched": false,
      "rating": 10,
      "description": "Space exploration",
      "imageUrl": "https://image.com/interstellar.jpg"
    }
    ```
  * Response: same object with `id`.

* **PUT /movies/{id}**

  * Request: (same as POST, but updates fields).
  * Response: updated object.

* **DELETE /movies/{id}**

  * Response: `204 No Content`.

### UI Pages/Sections:

* **Navbar**: Links â†’ Home, Add Movie. Include a **dark/light theme toggle** (left to right switch).
* **Footer**: Simple footer with copyright.
* **Home Page**: Display all movies in grid with poster image, title, rating, and a badge if watched/unwatched.
* **Movie Details Page**: Show full details of a selected movie.
* **Add Movie Page**: Form for adding a new movie.
* **Edit Movie Page**: Form pre-filled with existing movie details for editing.

### Components:

* **Header/Navbar** with theme toggle.
* **Footer**.
* **MovieCard**: Displays movie poster, title, rating, watched status.
* **MovieForm**: Reusable form for Add/Edit.

### Functionality:

* Use **fetch API** with async/await to call the backend.
* Store all API requests in `lib/api.js` helper file that reads from `.env.local`.
* Mobile responsive: grid collapses into single-column on small screens, navbar collapses into hamburger.
* Minimal animations: hover zoom on cards, button effects.

### Tech Stack:

* Next.js (App Router)
* TailwindCSS
* React Hooks
* Environment variables for API base URL
